# TripSync
A full-stack group trip companion — split expenses, chat in real time, manage checklists and get AI-powered suggestions.

## Tech Stack
- Phase 1: HTML, CSS, Vanilla JavaScript
- Phase 2: Firebase (Auth + Firestore + Realtime DB)
- Phase 3: Django + PostgreSQL + Gemini AI

## Status
🚧 Phase 1 in progress
